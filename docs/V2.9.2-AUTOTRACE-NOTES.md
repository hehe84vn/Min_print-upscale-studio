# V2.9.2 AutoTrace integration notes

AutoTrace is inherited as an external candidate engine for flat-color artwork. The app does not reimplement AutoTrace algorithms.

- Upstream: `autotrace/autotrace`
- Upstream release targeted by V2.9.2: `0.31.10`
- Program license: GPL-2.0-or-later
- `libautotrace` license: LGPL-2.1-or-later
- Current product scope: personal/internal use only
- Runtime policy: detect a packaged or system AutoTrace executable; if unavailable or unsuccessful, keep the VTracer result and record the fallback reason.
- Input format: PPM generated from Sharp raw RGB preprocessing, avoiding assumptions about optional ImageMagick decoders.
- Output format: SVG, normalized back to the source canvas/viewBox.
- Existing Geometry Lock and Binary Reconstruction remain fallback/reference code only and are not part of the primary V2.9.2 color path.
